# groot

## Project setup
#### Vue:
```
npm install
```
#### Python:

##### MacOS/ Linux
```
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
```
##### Windows
```
python -m venv venv
venv\Scripts\activate
pip install -r requirements.txt
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```

### start server
```
python entrypoint.py
```
